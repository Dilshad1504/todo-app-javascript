# Todo App

An interactive task management application built using JavaScript.

## 🚀 Features
- Add tasks
- Edit tasks
- Delete tasks
- Data stored in LocalStorage
- Responsive design

## 🛠 Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript
